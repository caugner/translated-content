Seit Firefox 33 wird ein "ev" Symbol im HTML Panel neben Elementen angezeigt, an die Event Listener gebunden sind:

Wird auf das Symbol geklickt, wird ein Popup angezeigt, das alle Event Listener anzeigt, die an dieses Element gebunden sind:

Jede Zeile beinhaltet:

  • einen Pausieren Button: Wird darauf geklickt, gelangt man zum Event Listener innerhalb des Debuggers, wo ein Haltepunkt für ihn gesetzt werden kann.
  • den Namen des Events.
  • den Namen und die Zeilennummer des Listeners: Wird darauf geklickt, wird die Listenerfunktion im Popup angezeigt.
  • die Information, ob ein Event nach oben steigt (Bubbling) und welche Spezifikation es definiert.

Beachte, dass auch jQuery Events untersucht werden können, nicht nur DOM Events.
