## Podsumowanie

W aplikacjach Mozilli `-moz-outline-radius-topleft` ustawia zaokrąglenie górnego lewego rogu konturu.

Zobacz więcej informacji we własności [-moz-outline-radius](pl/CSS/-moz-outline-radius).
