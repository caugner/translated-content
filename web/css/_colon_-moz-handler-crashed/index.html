Übersicht

:-moz-handler-crashed matcht Elemente, die nicht angezeigt werden können, weil das Plugin, das für deren Darstellung verantwortlich ist, abgestürzt ist.

Dies ist hauptsächlich für Themeentwickler nützlich.
