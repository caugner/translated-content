Übersicht

Das nicht standardisierte ::-moz-list-bullet Mozilla CSS Pseudoelement wird dazu benutzt, um den Aufzählungspunkt eines Listenelements zu stylen.

Syntax

li::-moz-list-bullet { style properties }

Beispiele

CSS

.list li::-moz-list-bullet {
  font-size: 36px;
}

HTML

<ul>
  <li>Number 1</li>
  <li>Number 2</li>
  <li>Number 3</li>
</ul>
<ul class="list">
  <li>Number 1</li>
  <li>Number 2</li>
  <li>Number 3</li>
</ul>
