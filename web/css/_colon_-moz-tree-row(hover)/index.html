Übersicht

Die :-moz-tree-row(hover) CSS Pseudoklasse matcht ein Element, falls der Mauszeiger sich über einer Baumzeile befindet.

Dieser Selektor ist hauptsächlich für die Verwendung durch Themeentwickler gedacht.
