## Podsumowanie

W aplikacjach Mozilli `-moz-outline-radius-bottomright` ustawia zaokrąglenie dolnego prawego rogu konturu.

Zobacz więcej informacji we własności [-moz-outline-radius](pl/CSS/-moz-outline-radius).
