## Übersicht

Das `::-moz-page` CSS [Pseudoelement](/de/docs/Web/CSS/Pseudo-elements) gilt für eine einzelne Seite im Druck oder der Druckvorschau.

## Spezifikationen

Nicht Teil einer Spezifikation.
