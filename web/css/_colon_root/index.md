### Podsumowanie

Pseudoklasa `:root` odnosi się do głównego węzła dokumentu.

### Składnia

    selektor:root {własności }

### Przykłady

    *:root { margin: 0; }

Przykład głównemu węzłowi dokumentu ustawi margines na zero.

### Notatki

W HTML4 głównym węzłem dokumentu jest zawsze element `HTML`.
