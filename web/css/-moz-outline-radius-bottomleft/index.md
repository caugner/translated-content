## Podsumowanie

W aplikacjach Mozilli `-moz-outline-radius-bottomleft` ustawia zaokrąglenie dolnego lewego rogu konturu.

Zobacz więcej informacji we własności [-moz-outline-radius](pl/CSS/-moz-outline-radius).
