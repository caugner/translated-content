Element `<xsl:comment>` zapisuje komentarz do dokumentu wyjściowego. Może zawierać tylko tekst.

### Składnia

    <xsl:comment>
    	SZABLON
    </xsl:comment>

### Wymagane atrybuty

Brak.

### Opcjonalne atrybuty

Brak.

### Typ

Instrukcja, pojawia się wewnątrz szablonu.

### Definicja

XSLT, sekcja 7.4.

### Gecko

Obsługuje.
