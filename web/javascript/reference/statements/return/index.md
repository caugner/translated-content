### Podsumowanie

### Składnia

`return wyrażenie;`

### Parametry

- `wyrażenie`
  - : wyrażenie, którego wynik ma być zwrócony.

### Przykłady

#### Przykład: Zastosowanie `return`

Poniższa funkcja zwraca kwadrat swojego argumentu `x` (gdzie `x` jest liczbą).

    function kwadrat(x) {
       return x * x;
    }
