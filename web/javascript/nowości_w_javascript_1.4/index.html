Funkcje, które zostały dodane w JavaScripcie 1.4 obejmują:

  • Dodano exception handling (throw oraz try-catch)
  • Dodano operatory in oraz instanceof
  • LiveConnect
  • zmiany eval (nie można używać pośrednio oraz usunięcie z metod Object)
  • arguments nie jest już właściwością funkcji
  • Metodę uznano za przestarzałą Function.arity na rzecz Function.length
