Übersicht

Das HTML <var> Element repräsentiert eine Variable. Es kann sich dabei um eine mathematischen Ausdruck handeln oder um eine Variable in einem Quellcode.

DOM Interface

Dieses Element implementiert das HTMLElement Interface.

Implementationshinweis: Bis einschließlich Gecko 1.9.2 implementiert Firefox das HTMLSpanElement Interface für dieses Element.

Attribute

Keine speziellen vorhanden, nur Universalattribute.

Standardmäßige Style-Eigenschaften

var {
  font-style: italic;
}

Beispiel

<p> Eine einfache Gleichung: <var>x</var> = <var>y</var> + 2 </p>

Eine einfache Gleichung: x = y + 2

Spezifikationen

  • HTML 4.01 var
  • HTML5 var
