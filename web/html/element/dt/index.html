Podsumowanie

Element <dt> (definition term) określa termin definicji w liście definicji (<dl>). Ten element może występować tylko jako dziecko listy definicji i powinien następować po nim element opis definicji (<dd>) element.

  • Dozwolona zawartość: liniowa
  • Dozwolony w: <dl>
