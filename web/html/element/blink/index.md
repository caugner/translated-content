## Podsumowanie

Element `<blink>...</blink>` powoduje migotanie zawartego w nim tekstu (działa tylko w IE, Operze i przeglądarkach opartych na silniku [Gecko](/pl/Gecko "pl/Gecko")).

## Przykład

```html
<blink>Why would somebody use this?</blink>
```
