## Podsumowanie

Element `<big>...</big>` powiększa czcionkę w stosunku do użytej treści.

## Przykład

```html
<p>
       To jest pierwsze zdanie.  <big>To zdanie jest napisane dużymi literami.</big>
</p>
```
