### Podsumowanie

**tFoot** zwraca element TFOOT tabeli.

### Składnia

    HTMLTableSectionElementObject = table.tFoot

### Przykład

    if (table.tFoot == my_foot) {
     ...
    }

### Uwagi

Ta własność zwraca VOID, jeśli nie istnieje element TFOOT.
