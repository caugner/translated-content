### Podsumowanie

**caption** zwraca podpis tabeli.

### Składnia

    string = table.caption

### Przykład

    if (table.caption) {
      // zrób coś z podpisem
    }

### Uwagi

Własność ta zwraca void dla tabel bez elementu caption.
