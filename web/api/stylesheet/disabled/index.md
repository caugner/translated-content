### Podsumowanie

Ta własność sygnalizuje, gdzie bieżący arkusz stylów jest używany lub nie.

### Składnia

    bool = stylesheet.disabled

### Przykład

    // jeśli arkusz stylów jest wyłączony...
    if (stylesheet.disabled) {
       // zachowuje style in-line
    }
