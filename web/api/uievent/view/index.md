### Podsumowanie

Atrybut **view** identyfikuje widok (interfejs AbstractView), w którym wygenerowane zostało zdarzenie.

### Składnia

    referencjaDoAbstractView = event.view

### Parametry

- `referencjaDoAbstractView` to rerefencja do obiektu AbstractView.
