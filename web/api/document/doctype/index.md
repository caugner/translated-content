### Podsumowanie

Zwraca definicję typu bieżącego dokumentu (DTD).

### Składnia

    ciąg = document.doctype

### Uwagi

Atrybut **doctype** jest tylko do odczytu. Jeżeli nie ma DTD w dokumencie, zwrócone zostaje NULL.
