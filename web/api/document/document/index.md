{{APIRef}}

Die **`Document`** Methode erstellt ein neues {{domxref("Document")}} Objekt das als Webseite im Browser geladen werden kann und bietet Möglichkeiten auf diese Inhalte zuzugreifen.

## Konstruktor

    new Document();
