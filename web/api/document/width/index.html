Podsumowanie

Zwraca szerokość bieżącego dokumentu w pikselach.

Nie obsługiwana przez IE.

Składnia

piksele = document.width

Przykład

function init() {
  alert("Szerokość dokumentu wynosi " + document.width + " pikseli.");
}

Uwagi

Zobacz także DOM:document.height.
