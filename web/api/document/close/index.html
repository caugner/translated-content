Podsumowanie

Metoda document.close() kończy zapis do otwartego dokumentu.

Składnia

document.close()

Przykład

// otwieramy dokument do zapisu
// i kończymy zamknięciem poprzez close()
document.open();
document.write("<P>The only content</P>.");
document.close();
