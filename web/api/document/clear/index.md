### Podsumowanie

Metoda **clear** czyści dokument, usuwając całą jego zawartość.

### Składnia

    document.clear()

### Przykład

    <button label="empty" onclick="document.clear();" />

### Specyfikacja

DOM Level 0 - brak w specyfikacji.
