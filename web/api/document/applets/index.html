Podsumowanie

applets zwraca uporządkowaną listę apletów umieszczonych w dokumencie.

Składnia

listaWęzłów = document.applets

Przykład

// (Kiedy wiesz, że drugi aplet jest tym,
// którego szukasz)
my_java_app = document.applets[1];
