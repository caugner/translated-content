### Podsumowanie

Własność **onkeydown** zawiera kod obsługi zdarzenia `keydown` dla bieżącego elementu.

### Składnia

    kod obsługi zdarzenia = element.onkeydown

### Uwagi

Zdarzenie `keydown` wywoływane jest, gdy użytkownik naciska klawisz na klawiaturze.
