Podsumowanie

enctype pobiera/ustawia typ zawartości elementu FORM.

Składnia

string = form.enctype
form.enctype =
string

Przykład

form.enctype = "application/x-www-form-urlencoded";

Uwagi

Typem kodowania jest ogólnie "application/x-www-form-urlencoded".
