#### Podsumowanie

length zwraca liczbę kontrolek w elemencie FORM.

#### Składnia

    integer = form.length

#### Przykład

    if (document.getElementById("form1").length > 1) {
      // więcej niż jedena kontrolka formularza jest tu
    }
