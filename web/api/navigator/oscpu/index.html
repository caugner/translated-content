Podsumowanie

Zwraca łańcuch znaków reprezentujący bieżący system operacyjny.

Składnia

oscpuInfo = window.navigator.oscpu
  • oscpu jest łańcuchem znaków przyjmującym poniższą formę.

Przykład

function osInfo() {
alert(window.navigator.oscpu);
}
// zwraca: Windows NT 6.0
