### Podsumowanie

Zwraca obiekt paska adresu okna.

### Składnia

    dirBar = window.directories

### Parametry

_dirBar_
jest obiektem typu `barProp`.

### Przykład

    <script>
     function dirs() {
      alert(window.directories);
     }
    </script>

### Specyfikacja

DOM Level 0. Nie jest częścią specyfikacji.
