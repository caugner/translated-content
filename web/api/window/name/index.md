### Podsumowanie

Pobiera/ustawia nazwę okna.

### Składnia

    string = window.name
     window.name =string

### Przykład

    window.name = "lab_view";

### Uwagi

Nazwa okna jest używana pierwotnie dla ustawienia celu dla linków oraz formularzy. Okno nie musi posiadać nazwy.

### Specyfikacja

DOM poziom 0. Nie jest częścią specyfikacji.
