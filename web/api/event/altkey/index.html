Podsumowanie

Wskazuje, czy klawisz <alt> był wciśnięty przy wywołaniu zdarzenia.

Składnia

bool = event.altKey

Przykład

 function goInput(e) {
 // sprawdza przycisk <alt>
   if e.altKey {
      // przekazuje zdarzenie dalej
      superSizeOutput(e);
   }
   else {
     doOutput(e);
   }
 }
