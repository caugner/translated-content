Podsumowanie

Zwraca poziomą współrzędną zdarzenia w obszarze klienta.

Przykład

 function checkClientClickMap(e) {
   if e.clientX < 50 {
     doRedButton(); }
   if 50 <= e.clientX < 100 {
     doYellowButton(); }
   if e.clientX >= 100 {
     doRedButton(); } }
