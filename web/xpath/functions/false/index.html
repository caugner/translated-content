Funkcja false zwraca wartość logiczną fałsz.

Składnia 

false()

Zwraca

Wartość logiczną false.

Uwagi

Ta funkcja jest użyteczna jako część porównania: 

<xsl:if test="boolean((1 &gt; 2) = false())">
 Wyrażenie jest ocenione jako true
</xsl:if>

Definicja

XPath 1.0 4.3

Gecko

Obsługuje.
