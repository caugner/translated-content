Funkcja `ceiling` ocenia liczbę dziesiętną i zwraca najmniejszą liczbę całkowitą większą lub równą podanej liczbie dziesiętnej.

### Składnia

    ceiling( liczba )

### Argumenty

- `liczba`
  - : Oceniana liczba.

### Zwraca

Najbliższą liczbę całkowitą większą lub równą `liczba`.

Przykładowo:

` ceiling (5.2)` = _6_

` ceiling (-5.2)` = _-5_

### Definicja
