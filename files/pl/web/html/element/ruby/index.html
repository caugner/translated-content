{{HTMLRef}}

Znacznik HTML <ruby> reprezentuje adnotację ruby. Adnotacje te są w celu wyświetlania wymowy znaków używanych w Azji Wschodniej.

{{EmbedInteractiveExample("pages/tabbed/ruby.html", "tabbed-shorter")}}

Źródło tego interaktywnego przykładu jest przechowywane na repozytorium GitHub. Jeżeli chciałbyś przyczynić się do interaktywnych przykładowych projektów, sklonuj, proszę https://github.com/mdn/interactive-examples i wyślij nam prośbę o dołączenie.

Kategorie zawartości Treść przepływu, treść wyrażona, treść oczywista.
Dozwolona treść Treść wyrażona.
Pominięcie znacznika {{no_tag_omission}}
Dozwoleni rodzice Każdy element akceptujący treść wyrażoną.
Dozwolone role ARIA Każda.
Interfejs DOM {{domxref("HTMLElement")}}

Atrybuty

Ten element zawiera tylko globalne atrybuty.

Przykłady

Przykład 1: Znak

<ruby>
  漢 <rp>(</rp><rt>Kan</rt><rp>)</rp>
  字 <rp>(</rp><rt>ji</rt><rp>)</rp>
</ruby>

Przykład 2: Wyraz

<ruby>
  明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
</ruby>
