Podsumowanie

Element <cite> zawiera cytowanie lub odwołanie do innych zasobów.

  • Typ elementu: inline
  • Dozwolona zawartość: inline

Przykład

Oznaczanie autora cytatu 

  As <cite>Harry S. Truman</cite> said, <q>The buck stops here.</q>

Powyższy HTML daje w efekcie:

Oznaczanie źródła 

  Więcej informacji można znaleźć w <cite>[ISO-0000]</cite>

Powyższy HTML daje w efekcie:

Więcej informacji można znaleźć w {{ mediawiki.external('ISO-0000') }}

Notatki

Aby usunąć kursywę z elementu <cite>, użyj własności CSS font-style.
