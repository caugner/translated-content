Podsumowanie

Element <b>...</b> wyróżnia poszczególne fragmenty tekstu, jednak nie nadaje im ważności sematycznej (tak jak robi to znacznik <strong>). Ze względu na zaszłość ze starymi wersjami HTMLa, znacznik nadaje pogrubienie dla tekstu, który ma wewnątrz.

Przykład

<p>
  To jest pierwsze wyrażenie.
  <b>To wyrażenie jest pogrubione.</b>
</p>

Rezultat

To jest pierwsze wyrażenie. To wyrażenie jest pogrubione.
