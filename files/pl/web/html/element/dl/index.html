Podsumowanie

Element lista definicji (<dl>) zawiera listę par terminów i ich definicji. Powszechnym zastosowaniem tego elementu jest tworzenie słownika.

  • Typ elementu: blokowy
  • Dozwolona zawartość: dt, dd

Atrybuty

compact {{ Non-standard_inline() }}
Wymusza, by opis definicji pojawił się w tej samej linii, co termin definicji. Działa tylko w Internet Explorer.

Przykłady

Pojedynczy termin i definicja

<dl>
    <dt>Firefox</dt>
        <dd>Wolna, o otwartym kodzie źródłowym, międzyplatformowa,
        graficzna przeglądarka internetowa,
        tworzona przez Mozilla Corporation i setki wolontariuszy.</dd>

        <!-- inne terminy i definicje -->
</dl>

Rezultat:

Firefox
Wolna, o otwartym kodzie źródłowym, międzyplatformowa, graficzna przeglądarka internetowa, tworzona przez Mozilla Corporation i setki wolontariuszy.

Wiele terminów, jedna definicja

<dl>
    <dt>Firefox</dt>
    <dt>Mozilla Firefox</dt>
    <dt>Fx</dt>
        <dd>Wolna, o otwartym kodzie źródłowym, międzyplatformowa,
        graficzna przeglądarka internetowa,
        tworzona przez Mozilla Corporation i setki wolontariuszy.</dd>

        <!-- inne terminy i definicje -->
</dl>

Rezultat:

Firefox
Mozilla Firefox
Fx
Wolna, o otwartym kodzie źródłowym, międzyplatformowa, graficzna przeglądarka internetowa, tworzona przez Mozilla Corporation i setki wolontariuszy.

Pojedynczy termin, wiele definicji

<dl>
    <dt>Firefox</dt>
        <dd>Wolna, o otwartym kodzie źródłowym, międzyplatformowa,
        graficzna przeglądarka internetowa,
        tworzona przez Mozilla Corporation i setki wolontariuszy.</dd>
    <dd>Panda czerwona, znana też jako panda mniejsza, Wah, Bear Cat lub Firefox,
        jest to przeważnie trawożerny ssak, nieco większy niż kot domowy
        (60 cm długości).</dd>

    <!-- inne terminy i definicje -->
</dl>

Rezultat:

Firefox
Wolna, o otwartym kodzie źródłowym, międzyplatformowa, graficzna przeglądarka internetowa, tworzona przez Mozilla Corporation i setki wolontariuszy.
Panda czerwona, znana też jako panda mniejsza, Wah, Bear Cat lub Firefox, jest to przeważnie trawożerny ssak, nieco większy niż kot domowy (60 cm długości).

Wiele terminów i definicji

Możliwe jest również tworzenie wielokrotnych terminów i definicji, ale już wiesz, jak to zrobić.

Notatki

Użycie <dl compact> jest poprawne w HTML, lecz, aby użyć tego atrybutu w XHTML, musisz użyć zapisu <dl compact="compact">.

Nie zaleca się używania tego elementu, razem z niesortowanymi listami, żeby stworzyć tylko wcięcie na stronie. Mimo, iż to działa, jest to zła praktyka i pokazuje złe rozumienie znaczenia listy definicji.

Aby zmienić wcięcie terminu definicji, użyj własności CSS margin.
