Podsumowanie

Zwraca wartość logiczną wskazującą, czy klawisz <meta> był wciśnięty podczas wywołania zdarzenia.

Przykład

 function goInput(e) {
 // sprawdza metaKey
   if (e.metaKey) {
        // i przekazuje zdarzenie dalej
     superSizeOutput(e);

    } else {
     doOutput(e);
    }
 }

Uwagi

Na niekótrych platformach metaKey może obsługiwać klawisz o innej nazwie.
