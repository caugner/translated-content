Podsumowanie

hasAttributes zwraca wartość logiczną zależnie od tego, czy element posiada jakiekolwiek atrybuty.

Składnia

[ true | false ] = element.hasAttributes

Przykład

t1 = document.getElementById("table-data");
if ( t1.hasAttributes ) {
    // zrób coś z
    // t1.attributes
}
