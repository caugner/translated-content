Podsumowanie

target pobiera/ustawia element docelowy działania (np., w której ramce ma być wczytana zawartość strony).

Składnia

string = form.target
form.target =
string

Przykład

myForm.target = document.frames[1].name;

Specyfikacja

DOM Level 2 HTML: target
