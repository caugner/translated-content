Podsumowanie

tHead zwraca THEAD (nagłówek) tabeli.

Składnia

th_el = table.tHead

Parametry

  • th_el jest HTMLTableSectionElement.

Przykład

if (table.tHead == my_head_el) {
   ...
}

Uwagi

Ta własność zwraca VOID, jeśli nie istnieje element THEAD.
