Podsumowanie

Zwraca URL bieżącego dokumentu.

Składnia

ciąg = document.URL

Przykład

var currentURL = document.URL;
alert(currentURL);

Uwagi

URL ma zastąpić używaną w DOM Level 0 własność document.location.href. Jednakże, w przeciwieństwie do niej, document.URL nie można ustawiać.
