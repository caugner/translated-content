Podsumowanie

Wskazuje, czy zdarzenie bąbelkuje przez model dokumentu czy nie.

Składnia

bool = event.bubbles

Przykład

 function goInput(e) {
  // sprawdza czy zdarzenie bąbelkuje
  if not e.bubbles {
     // i jeśli nie - przekazuje je dalej
     passItOn(e);
  }
  // jeśli bąbelkuje:
  doOutput(e)
}
