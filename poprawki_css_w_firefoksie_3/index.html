W programie Firefox 3 wprowadzono szereg udoskonaleń obsługi arkuszy stylów CSS. Poszczególne zmiany są przedstawione na odpowiednich stronach z dokumentacją — niniejszy artykuł zawiera jedynie krótkie omówienie umożliwiające szybkie zapoznanie się z nowościami.

  • Obsługiwane są wartości inline-block i inline-table dla własności {{ Cssxref("display") }}.
  • Własność {{ Cssxref("font-size-adjust") }} jest teraz obsługiwana na wszystkich platformach (poprzednio tylko w systemie Windows).
  • Obsługiwane są wartości rgba() i hsla() dla własności {{ Cssxref("color") }} ({{ Bug(147017) }}).
  • Obsługiwana jest pseudoklasa {{ Cssxref(":default") }} ({{ Bug(302186) }}).
  • Dla własności {{ Cssxref("width") }}, {{ Cssxref("min-width") }} i {{ Cssxref("max-width") }} można teraz użyć wartości -moz-max-content, -moz-min-content, -moz-fit-content i -moz-available ({{ Bug(311415) }} i {{ Bug(402706) }}).
  • Obsługiwane są łączniki opcjonalne w języku HTML (&shy;).
  • Znaki tabulacji w tekście preformatowanym są teraz lepiej obsługiwane w przypadku użycia czcionek proporcjonalnych (zgodnie ze specyfikacją CSS 2.1).
  • Obsługiwana jest własność {{ Cssxref("ime-mode") }}.
  • Obsługiwana jest własność CSS text-rendering dla treści w języku HTML ({{ Bug(387969) }}).
  • Zaimplementowano obsługę własności CSS -moz-border-*-start i -moz-border-*-end ({{ Bug(74880) }}).
  • Wartość -moz-initial jest obsługiwana przez niemal wszystkie własności CSS (z wyjątkiem znaczników quote i wartości -moz-border-*-color); patrz {{ Bug(80887) }}.
  • Metoda window.getComputedStyle() zwraca wszystkie obsługiwane własności CSS ({{ Bug(316981) }}).
  • Obsługiwana jest wartość none dla własności {{ Cssxref("content") }} ({{ Bug(378535) }}).
