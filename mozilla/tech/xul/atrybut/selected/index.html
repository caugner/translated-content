selected
Typ: boolean
Sygnalizuje gdzie element jest zaznaczony lub nie. Ta wartość jest tylko do odczytu. Aby zmienić zaznaczenie, ustaw pozostałe własności selectedIndex lub selectedItem stanowiące ten element.
