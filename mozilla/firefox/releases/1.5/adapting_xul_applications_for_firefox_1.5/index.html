Strona zawiera listę zmian w Firefoksie 1.5 wpływających na pracę programistów XUL.

Zmiany

  • Zmiany w drzewie API
  • Międzynarodowe pismo w XUL JavaScript (wpływa wyłącznie na rozszerzenia z plikami JavaScript zawierającymi znaki non-ASCII)
  • Zmiany w XMLHttpRequest
  • Zmiany w XUL-u w Firefoksie 1.5
  • XPCNativeWrapper są domyślnie włączone, a ich działanie zmienione na podobne do działania w wersji 1.0.x
  • Prostsza metoda rejestracji Chrome przestarzałego contents.rdf.
  • Dla nakładek menu kontekstowego: nazwa funkcji <tt>gContextMenu.linkURL()</tt> została zmieniona na <tt>gContextMenu.getLinkURL()</tt>, a <tt>linkURL</tt> jest teraz własnością. Aby użyć w trybie zgodności wstecznej:
    <tt>url = 'getLinkURL' in gContextMenu ? gContextMenu.getLinkURL() : gContextMenu.linkURL();</tt>
