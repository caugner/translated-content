| Typ         | `Number (numer)` |
| ----------- | ---------------- |
| Obowiązkowy | Tak              |
| Przykład    | ```json          |

"manifest_version": 2

````|

Ten klucz określa wersję pliku manifest.json użytego w rozszerzeniu.

Obecnie zawsze musi to być 2.

## Przykład

```json
"manifest_version": 2
````
